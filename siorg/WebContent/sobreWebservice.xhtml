A URL padrão para acessar as funcionalidades é : ../rest/service/{funcao},

onde {funcao} é o nome da função (Consulte a aba "Funções" para mais detalhes) URL : ../rest/service/completo

Recupera a estrutura completa com Órgãos, Servidores e Cargos URL : ../rest/service/todosServidores

Recupera todos Servidores URL : ../rest/service/todosOrgaos

Recupera todos Órgãos URL : ../rest/service/todosCargos

Recupera todos Cargos URL : ../rest/service/servidoresPorCargo/{cargo}

Recupera Servidores por Cargo, onde {cargo} é o nome do cargo URL : ../rest/service/servidoresPorOrgao/{orgao}

Recupera Servidores por Orgão, onde {orgao} é o nome do Órgão URL : ../rest/service/servidoresPorMatricula/{matricula}

Recupera Servidor por Matrícula, onde {matricula} é a matrícula Pode recuperar os dados via URL ou via Classe Java Segue um exemplo de classe que pode usar para utilizar o serviço

public void getTodosXML() {
ClientConfig config = new DefaultClientConfig();
Client client = Client.create(config);
WebResource service = client.resource(URL_BASE);
service.path("rest").path("service").path("funcao").accept(MediaType.TEXT_XML).get(String.class));
}
